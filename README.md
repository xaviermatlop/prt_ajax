# Práctica mini_crud_ajax

He construido una web copiando y pegando donde la interacción es instantánea porque las acciones suceden sin recargar la página.

fetch(): He aprendido a usar fetch() en JavaScript para que mi navegador se habla con el servidor (PHP) en segundo plano. Esto me permite enviar y recibir datos como agregar o borrar usuarios de forma asíncrona.

Contrato de Datos (JSON): He entendido que mi servidor no me devuelve HTML en su lugar, me responde paquetes de datos en formato JSON, el código JavaScript toma ese JSON para actualizar solo la parte necesaria de la pantalla como la tabla.
